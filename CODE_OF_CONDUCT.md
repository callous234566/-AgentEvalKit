# Code of Conduct / 行为准则

This project follows a simple contributor code of conduct.

## Our Standards / 我们鼓励

- Be respectful and constructive.
- Discuss ideas and tradeoffs, not personal attributes.
- Keep feedback specific and actionable.
- Avoid sharing private data, credentials, or confidential documents.

## Unacceptable Behavior / 不接受

- Harassment, insults, or discriminatory language.
- Publishing someone else's private information.
- Posting secrets, private documents, or production credentials.
- Deliberately disruptive or misleading contributions.

## Enforcement / 执行

Maintainers may edit, hide, or remove comments and contributions that violate this code of conduct. Repeated or severe violations may result in blocked participation.
